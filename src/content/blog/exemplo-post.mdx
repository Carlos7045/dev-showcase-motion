# Como Configurar um Projeto React com TypeScript e Vite

Neste tutorial, vamos aprender como configurar um projeto React moderno usando TypeScript e Vite, duas ferramentas essenciais para desenvolvimento frontend em 2024.

## Por que usar Vite?

Vite é um build tool extremamente rápido que oferece:

- **Hot Module Replacement (HMR)** instantâneo
- **Build otimizado** para produção
- **Suporte nativo** ao TypeScript
- **Configuração mínima** necessária

## Configuração Inicial

Primeiro, vamos criar um novo projeto:

```bash
npm create vite@latest meu-projeto -- --template react-ts
cd meu-projeto
npm install
```

### Estrutura de Pastas Recomendada

```
src/
├── components/
│   ├── ui/
│   └── layout/
├── hooks/
├── utils/
├── types/
└── pages/
```

## Configurando o TypeScript

O arquivo `tsconfig.json` deve incluir as seguintes configurações:

```json
{
  "compilerOptions": {
    "target": "ES2020",
    "useDefineForClassFields": true,
    "lib": ["ES2020", "DOM", "DOM.Iterable"],
    "module": "ESNext",
    "skipLibCheck": true,
    "moduleResolution": "bundler",
    "allowImportingTsExtensions": true,
    "resolveJsonModule": true,
    "isolatedModules": true,
    "noEmit": true,
    "jsx": "react-jsx",
    "strict": true,
    "noUnusedLocals": true,
    "noUnusedParameters": true,
    "noFallthroughCasesInSwitch": true
  },
  "include": ["src"],
  "references": [{ "path": "./tsconfig.node.json" }]
}
```
